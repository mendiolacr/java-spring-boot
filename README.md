# Java Spring Boot

## Installation 🔧

Clone the project
```bash
git clone build
```

```bash
docker-compose up -d
```

##### Functionalities
- Save gateway and validate IPv4
- Save devices and validate that each gateway does not accept more than 10 devices
- Show all gateways
- Show gateway
- Delete device
